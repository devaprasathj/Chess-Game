# ♟️ Chess Game

A lightweight, modern, and interactive browser-based Chess Game built completely with vanilla web technologies. The application features a clean user interface, responsive game logic, and smooth animations, providing an engaging experience directly within your web browser.

Live Link: https://chess-board-rnza.onrender.com

---

## 🚀 Features

* **Complete Chess Rules:** Supports standard chess piece movements, turns, and basic game loops.
* **Interactive Visual Interface:** Highlights available moves when a piece is selected for intuitive gameplay.
* **Responsive Layout:** Fully optimized for seamless play on both desktop screens and mobile viewports.
* **Zero Dependencies:** Built entirely with native web technologies for fast loading times and optimal performance.

---

## 🛠️ Tech Stack

* **HTML5:** Structured layout representing the 8x8 chess grid and interactive elements.
* **CSS3:** Custom styles for alternating board squares, typography, hover animations, and responsiveness.
* **JavaScript (ES6+):** Pure vanilla JavaScript handling piece selection, move validation, array coordinates, and game state logic.
